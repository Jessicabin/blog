# 批量引入自定义组件

在业务代码中时常会自行封装一些基础组件，如table、form等，使用频率高，如果每次在页面中引用不方便，需要全局引入这些组件。

创建一个js文件，引入vue及自定义组件，使用Vue.component()方法挂载，并在main.js中引入该文件。

在components.js 中引入所需组件

import Vue from 'vue'
import BaseTable from '@/components/base-table'
import BaseForm from '@/components/base-form'

//定义一个对象，使用for...of批量引入自定义组件,减少重复代码
const components = {
  BaseTable, BaseForm
}

for (const [key, value] of Object.entries(components)) {
  Vue.component(key, value)
}

在main.js 引入 components.js，即可全局调用这些组件

import './plugins/components'
