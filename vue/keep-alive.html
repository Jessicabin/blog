# keep-alive

<keep-alive>
    <router-view></router-view>
</keep-alive>

用keep-alive包裹动态组件时，它会缓存不活动的组件实例。

因此它只会执行一次created、mounted（即数据不会进行相应刷新，除非手动触发），且在跳转到下一个keepAlive之前，才会执行beforedestroy

# 生命周期执行规则

首次执行该keep-alive组件

进入：beforeCreate -> created -> beforeMount -> mounted -> activated 离开：deactivated

第二次执行该keep-alive组件

进入：activated 离开：deactivated

没有keep-alive的组件(Vue实例生命周期)

进入：beforeCreate -> created -> beforeMount -> mounted 离开：beforeDestroy -> destroyed

离开当前路由销毁页面内keep-alive组件

如果当前路由已经由keep-alive包裹，且当前路由下又有keep-alive组件，离开当前页面时想销毁已有的keep-alive组件，回到初始状态，可用以下方法。

beforeRouteLeave (to, from, next) {
  if (this.$vnode && this.$vnode.data.keepAlive && this.$vnode.parent) {
      const tag = this.$vnode.tag;
      let caches = this.$vnode.parent.componentInstance.cache;
      let keys = this.$vnode.parent.componentInstance.keys;
      for (let [key, cache] of Object.entries(caches)) {
          if (cache.tag === tag) {
              if (keys.length > 0 && keys.includes(key)) {
                  keys.splice(keys.indexOf(key), 1);
              }
              delete caches[key];
          }
      }
  }
  this.$destroy();
  next();
},
