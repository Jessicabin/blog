# flex及伸缩布局应用

一般展示大屏首页会分为header、nav和main三块区域，main则包括左中右三部分，左右面板可伸缩。

整体布局样式使用css flex，并通过设定宽度达到可伸缩效果。

主要代码如下：

<div class="page-wrapper">
  <div class="header"></div>
  <div class="nav"></div>
  <div class="main">
    <div class="screen-wrapper">
      <div class="left-wrapper" :class="{ hide: hidden.left }"></div>
      <div class="center-wrapper">
        <div class="fold-icon left-icon" :class="getIconStyle('left')" @click="setHidden('left')" />
        <div class="fold-icon right-icon" :class="getIconStyle('right')" @click="setHidden('right')" />
        <div class="center-container">
        </div>
      </div>
      <div class="right-wrapper" :class="{ hide: hidden.right }">
      </div>
    </div>
  </div>
</div>  

export default {
  data() {
    return {
      hidden: {
        left: false,
        right: false,
      },
    }
  },
  computed: {
    getIconStyle() { //icon图标
      return (type) => {
        let map = {
          left: { false: 'unfold-left', true: 'fold-left' },
          right: { false: 'unfold-right', true: 'fold-right' },
        }
        return map[type][this.hidden[type]]
      }
    },
  },
  methods: {
    setHidden(type) { //控制左右两侧面板显隐
      this.hidden[type] = !this.hidden[type]
    },
  },
}

页面整体布局使用flex，header和nav固定宽度，main占满剩余空间。

收缩按钮放在main区域的center中。

.page-wrapper {
  width: 100vw;
  height: 100vh;
  display: flex;
  flex-direction: column;
  .header {
    height: 50px;
  }
  .nav {
    height: 50px;
  }
  .main {
    flex: 1; 
    overflow: hidden;
    .screen-wrapper { //占满元素空间
      width: 100%;
      height: 100%;
      display: flex;
      .left-wrapper,
      .right-wrapper {
        width: 200px; //指定宽度
        height: 100%;
      }
      .center-wrapper {
        flex: 1;
        padding: 0 16px;
        position: relative;
        .center-container {
          width: 100%;
          height: 100%;
        }
        .fold-icon { //伸缩按钮样式，居中
          position: absolute;
          z-index: 99;
          cursor: pointer;

          &.left-icon,
          &.right-icon {
            top: 50%;
            transform: translateY(-50%);
            width: 16px;
            height: 100px;
          }
          &.left-icon {
            left: 0;
          }
          &.right-icon {
            right: 0;
          }
        }
      }
    }
  }
}

伸缩样式

通过设定左右两侧收起时宽度为0，overflow为hidden，以及加上变化的效果，达到伸缩效果。

.left-wrapper,
.right-wrapper {
  overflow: hidden;
  &.hide {
    width: 0;
  }
}
.center-wrapper {
  overflow: hidden;
  .center-container {
    overflow: hidden;
  }
}
.left-wrapper,
.right-wrapper,
.center-wrapper,
.fold-icon {
  transition: all 0.5s linear;
}
